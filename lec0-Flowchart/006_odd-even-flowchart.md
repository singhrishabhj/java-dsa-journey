# Odd Even Flowchart 

## Problem

Check whether a number is Odd or Even.

---

## Logic

A number is:

* Even if divisible by 2
* Odd if not divisible by 2

---

## Steps (Algorithm Thinking)

1. Start
2. Input number N
3. Check N % 2 == 0

   * Yes → Even
   * No → Odd
4. Display result
5. End
